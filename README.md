# Mechatronics Exam Study Assistant

A modern Android application built with Jetpack Compose, featuring an interactive home screen widget designed to help students prepare for their final engineering exam.

# Asset Files & Customization

The `app/src/main/assets/` directory contains three data files:
* `arkusz1.txt`
* `arkusz2.txt`
* `arkusz3.txt`

These files currently contain official exam questions and topics for the **Mechatronics** major defense exam.

### How to use it for your own exam:
The application is fully customizable. To adapt the question database to your own needs or a different field of study, simply **replace these three files** with your own text files, keeping the exact same naming convention and formatting structure.

# Tech Stack

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Component:** Android AppWidget API (Optimized for Android 12+)
* **Minimum SDK:** API 31 (Android 12)
* **Target SDK:** API 34 (Android 14)
* **Build System:** Gradle (Kotlin DSL)
